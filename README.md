# Laravel Tips

```
Usage: laraveltips [-v] [<command>] [<args>]

A command line tool for laravel tips

Options:
  -v, --version     show version
  --help            display usage information

Commands:
  random            random laravel tips
  sync              sync laravel tips from laravel docs
```
